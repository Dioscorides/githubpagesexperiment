# Welcome to DMMapp Library Directory

Welcome to the documentation for the DMMapp Library Directory, a curated collection of digitized manuscript libraries from around the world.

## Overview

The DMMapp Library Directory provides a comprehensive database of libraries that offer digitized medieval manuscripts. Our platform helps researchers, historians, and enthusiasts discover and access these valuable cultural resources.

1. Visit the [Dashboard](/index.html) to explore the library directory
2. Use the search and filter tools to find specific collections
3. Click "Visit" to access each library's digitized manuscripts

## Data Schema

Each library entry includes:

- **Library Name**: Official name of the institution
- **Location**: City and nation
- **Website**: Direct link to the digitized collection
- **IIIF Support**: Whether the library implements IIIF standards
- **License**: Information about access restrictions and reuse rights
- **Quantity**: Approximate number of digitized manuscripts (Few, Dozens, Hundreds, Thousands, Unknown)

## About IIIF

The [International Image Interoperability Framework](https://iiif.io/) (IIIF) is a set of open standards for delivering high-quality, attributed digital objects online at scale. Libraries that support IIIF enable better discovery, comparison, and manipulation of their digital collections.
